Theodore's Lair is a mod that aims to make Diablo 1 Hellfire great again.

The mod adds a larger difficulty increase between each difficulty, adds two new classes: The Cleric and The Bomoh, adds a new helpful NPC and a few new items.

We also host a gaming network, a new off-game trading system and a character database.
